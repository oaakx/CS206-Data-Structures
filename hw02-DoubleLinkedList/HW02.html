public void removeNegative()

It removes negative elements from a sequence. The order of remained sequence is same as the order of the input sequence. Set cursor to new head.

Example

[8 20 -3 48 12 -1] -> [8 20 48 12]

Precondition

The number of elements is larger than 1.

public void removeMostDuplicate()

It removes elements whose number is duplicated the most times. If there is no unique number which is duplicated the most times, it removes nothing. The order of remained sequence is same as the order of the input sequence. Set cursor to new head.

Example

[-1 1 -1 2 4 2 -1 10 -13] -> [1 2 3 2 10 -13]

[-1 1 -1 2 4 2] -> [-1 1 -1 2 4 2]

Precondition

The number of elements is larger than 1.
