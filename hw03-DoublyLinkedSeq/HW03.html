Submit the implementation of the DoubleLinkedSeq class through Elice system. You have to fil out the methods in DoublyLinkedSeq.java and Node.java file in the Elice system. In the test, we are not testing Node.java file, but you will need this file to implement DoublyLinkedSeq.

For the details, please read the document in KLMS.

  • public void advanceNstepForward(int n)
* move cursor forward to the n steps.
* Example
  * ['h','e','l','l','o'], current = 0, getCurrent(): 'h'; 
  * advanceNstepForward(4);
  *  -> current = 4, getCurrent(): 'o'
* Parameter
    * n - cursor move to n steps forward if possible
* Precondition
    * isCurrent() returns true.
    * There are element at current cursor position + n steps. 
* Throws: IllegalStateException
    * indicate that
    - 1) there is no current element. 
    - 2) there is no element at target position, so advance may not be called and current cursor must be remained at the original position.
  • public void advanceNstepBackward(int n)
* similar to the advanceNstepForward(int n), 
* move cursor "backward" to the n steps.
* Parameter
    * n - cursor move to n steps backward if possible
* Precondition
    * isCurrent() returns true.
    * There are element at current cursor position - n steps. 
* Throws: IllegalStateException
    * indicate that
    - 1) there is no current element. 
    - 2) there is no element at target position, so advance may not be called and current cursor must be remained at the original position.
