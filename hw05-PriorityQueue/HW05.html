In this homework, you are able to practice queue data structure.

  1. Implement the PQDirect class and PQOrdinary class according to the specification given in Figure 7.12 in the textbook.

PQDirect

You can implement this class by implementing your own Node class

PQOrdinary

You can implement this class by using ArrayQueue class. Please don’t erase queues parameter and getQueues() method.

  1. Implement four key methods of Double-ended queue class

Deque

You implement the additional methods of the inheritor, using the variables of the super class with the keyword “super.” Please do not make the identical variable to those of the ArrayQueue, such as data, manyItems, front, and rear. Except those, you can make any method and variable.

  • public void addFirst(E element) Insert a new item at the front of this queue. If the addition would take this queue beyond its current capacity, then the capacity is increased before adding the new item. The new item may be the null reference.
  • param element The element to be pushed at the front of queue
  • Postcondition The item has been pushed onto this queue.
  • Exception OutOfMemoryError Indicates insufficient memory for increasing the queue’s capacity
  • Note An attempt to increase the capacity beyond
  • public void addLast(E element) Insert a new item at the rear of this queue. If the addition would take this queue beyond its current capacity, then the capacity is increased before adding the new item. The new item may be the null reference.
  • param element The element to be pushed at the rear of queue
  • Postcondition The item has been pushed onto this queue.
  • Exception OutOfMemoryError Indicates insufficient memory for increasing the queue’s capacity
  • Note An attempt to increase the capacity beyond
  • public E removeFirst() Get the front item, removing it from this queue.
  • Precondition This queue is not empty.
  • Return The return value is the front item of this queue, and the item has been removed.
  • Exception NoSuchElementException Indicates that this queue is empty.
  • Note The front item means the first input using the add method of the ArrayQueue class or the latest input of addFirst method of the Deque.
  • public E removeLast() Get the last item, removing it from this queue.
  • Precondition This queue is not empty.
  • Return The return value is the last (rear) item of this queue, and the item has been removed.
  • Exception NoSuchElementException Indicates that this queue is empty
  • Note The last item means the latest input using the add method of the ArrayQueue class or the addLast method of the Deque.
